# 尼泊尔徒步行程篇

## 这部分对应哪些原始笔记

这页主要吸收了“每日行程安排（含住宿、交通与徒步）”“ABC 冲顶日完整版攻略”“什么是 ABC 冲顶日？”“昆明中转方案”“11天行程（含9天尼泊尔徒步）”等原始内容。

## 行程写法的核心

真正有用的行程，不只是写 D1 到 D10，而是要把这些信息一起写清楚：

- 每天的区间目标
- 海拔变化
- 行走时长
- 住宿节点
- 补给和缓冲
- 哪一天最容易出问题

## 对大多数人的实用建议

- 把前两天当成进入节奏，而不是急着证明状态
- 把冲顶日前一天当成关键准备日
- 把回撤日也当成正经行程，别因为“快结束了”就放松判断

## 后续要继续补的内容

- 每日海拔折线图
- 雨天行程调整方案
- 关键节点的住宿与补给说明
- 从香港或内地主要城市出发的转场模板
