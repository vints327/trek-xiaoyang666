# ABC + Poon Hill

## 为什么推荐这条线

- 11 天左右可以完成，假期友好
- 风景密度高，覆盖日出、雪山、森林和村落
- 相比更高海拔线路，难度和风险更可控
- 对第一次去尼泊尔徒步的人更友好

## 核心特征

- 推荐季节：10 到 11 月、3 到 5 月
- 体验关键词：雪山视野、茶屋徒步、连续爬升、节奏管理
- 主要风险：高反、降雨、保暖不足、补水与恢复不足

## 出发前重点

- 不要只看海拔，还要看连续天数和每日爬升
- 住宿、补给、天气和下撤节奏要一起考虑
- 行程设计要保守，给天气与体能波动留余量
