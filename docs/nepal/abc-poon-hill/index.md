# ABC + Poon Hill

## 这条线为什么适合做发布版核心

在你现有原始笔记里，ABC + Poon Hill 的内容最完整：有路线规划、温度与天气、英文简称、补给、住宿、装备、拍摄、财物安全、打包策略和详细行程。它已经不是零散记录，而是可以真正生长成专题站的内容骨架。

## 路线气质

这条线兼顾风景密度和完成度。你既能看到经典日出与雪山全景，也能进入高海拔徒步的真实节奏：连续爬升、补给管理、保暖判断、住宿取舍和体能分配。

## 这条线最适合谁

- 第一次去尼泊尔，想走一条完整中长线的人
- 有 9 到 11 天完整假期的人
- 想在“难度、风景、预算、成熟度”之间找平衡的人
