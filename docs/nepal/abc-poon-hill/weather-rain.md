# ABC 雨季与天气应对

## 这页对应哪些原始笔记

- `ABC 徒步（含 Poon Hill）的温度范围`
- `ABC徒步（4.25-5.5）下雨几率与应对`

## 为什么天气页值得单独写

因为很多人真正卡住的不是“能不能走”，而是“这种天气下还值不值得继续按原计划走”。在 ABC 这种连续多日路线里，天气不是背景，而是行程的一部分。

## 下雨意味着什么

- 体感温度下降
- 路面湿滑，节奏被迫放慢
- 装备更容易失效
- 休息时更容易冷下来
- 拍摄计划和能见度都会受影响

## 实用原则

- 把雨天当作独立场景处理
- 服装、防水和节奏要同时调整
- 不要靠意志硬顶恶劣天气
- 能降级就降级，能缓冲就缓冲

## 最常被低估的点

下雨本身未必最危险，真正危险的是“湿 + 冷 + 累 + 判断下降”同时发生。
