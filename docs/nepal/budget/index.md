# 尼泊尔徒步预算篇

## 这部分对应哪些原始笔记

这页吸收了“尼泊尔旅行花费预估与路线规划”“机票中包含的600多元税费”等原始笔记，把预算从单次方案整理成更稳定的框架。

## 预算不是一个数字，而是一组变量

你最后花多少钱，通常受这些因素影响：

- 国际机票价格波动
- 境内转场选择
- 是否请向导或背夫
- 是否临时补租装备
- 住宿和补给的舒适度取向
- 是否给延误和应急留缓冲

## 预算的正确拆法

- 基础必需成本
- 行程舒适升级成本
- 装备采购或租赁成本
- 风险缓冲成本

## 为什么预算页很重要

很多公开攻略只给一个总价，看起来很爽，但对真正要出发的人帮助不大。真正有价值的是知道哪些钱不能省，哪些钱可以根据自己的目标调整。
