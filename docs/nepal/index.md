# 尼泊尔徒步

这部分内容来自原始笔记里的尼泊尔专题、ABC + Poon Hill 线规划攻略、背景信息、装备与着装篇、拍摄记录和行程安排。发布版的目标，是把这些原本分散的笔记整理成一条读者能顺着看下去的路线。

## 这部分站点的阅读逻辑

先理解尼泊尔为什么适合做中长线徒步，再选路线，再看季节与许可，最后再进入 ABC + Poon Hill 的装备、行程、预算和拍摄细节。

## 你可以从这里开始

- [路线选择](/nepal/routes/)：先判断自己该走哪类线
- [背景与目的地](/nepal/background/)：理解尼泊尔徒步为什么独特
- [季节与许可](/nepal/season-permits/)：解决最实际的准备问题
- [ABC + Poon Hill](/nepal/abc-poon-hill/)：进入具体线路

## 当前发布版重点

目前先以 ABC + Poon Hill 为核心，因为它最适合从内部笔记转成一套完整的公开专题：线路成熟、内容密度高、同时兼顾新手友好和风景密度。
