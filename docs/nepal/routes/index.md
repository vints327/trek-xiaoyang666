# 尼泊尔徒步路线选择

## 为什么先谈路线，而不是先谈梦想

很多人一上来就会盯着 EBC、ABC、大环线这些名字，但路线选择真正要回答的是：你的假期、预算、体能、高海拔经验和风险承受度，分别到什么程度。

## 常见三类路线

### 轻体验型
适合假期有限、想兼顾城市与轻徒步的人。重点在于体验山地氛围，而不是追求高海拔或长距离。

### 经典中长线
代表是 ABC + Poon Hill。它的优势是风景密度高、天数相对友好、补给和住宿成熟，是很多人第一次尼泊尔徒步的优先选择。

### 高海拔挑战型
比如 EBC 或更长的大环线。这类路线更依赖海拔适应、节奏控制和经验基础，不适合把“梦想值”直接等同于“适合自己”。

## 选路线时最该看什么

- 有多少完整假期
- 能接受多高海拔
- 是否有连续多天徒步经验
- 更在乎风景密度，还是更在乎挑战感
- 是否愿意为向导、背夫、缓冲天数买单

## 对多数第一次去的人

如果你只有 9 到 11 天，又想要一条完成度高、可读性强、公开分享也容易讲清楚的线路，那么 ABC + Poon Hill 基本是最平衡的选择。
