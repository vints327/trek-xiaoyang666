# 尼泊尔徒步装备篇

## 这部分对应哪些原始笔记

这页主要吸收了“关于装备与着装篇”里的材料，包括：最终精简装备、一次性雨衣、充电宝、医药小药包、头灯、睡袋内胆、便携小包、水袋和保温杯、登山杖与护膝、防水收纳、高海拔防晒等。

## 打包的核心原则

- 先保证保暖、防水、补水和照明
- 再考虑舒适性和拍摄需求
- 不拿高海拔环境做首次装备测试
- 每个物品都要有场景理由

## 穿着层次

尼泊尔徒步最大的服装问题不是“穿得多不多”，而是行进中、休息时、早晚温差中的切换效率。真正好用的系统，必须兼顾排汗、保暖和防风防雨。

## 很容易低估的装备

- 头灯
- 充电宝与充电线管理
- 睡袋或睡袋内胆
- 防水收纳
- 水袋或水壶系统
- 墨镜、防晒和唇部护理

## 你应该在出发前完成的事

- 按实际负重做一次徒步测试
- 完成一次完整打包
- 把高频拿取物品放在最顺手的位置
- 明确哪些能租、哪些一定要自带
