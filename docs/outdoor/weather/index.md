# 天气判断

## 很多人低估天气的方式

新手最常见的误区，是把天气理解成“温度”。但在户外场景里，真正影响体验和风险的，通常是温度、风、降雨、湿度、紫外线和能见度的组合。

## 你至少要一起看这些因素

- 温度和体感温度
- 风力和风口地形
- 是否降雨，以及降雨持续时间
- 湿度和地面湿滑程度
- 紫外线强度
- 云层与能见度

## 不同天气意味着什么

### 高温
更容易脱水、暴晒和节奏崩掉。看起来好走，但实际上容易因为补水不足和中暑导致状态断崖式下降。

### 低温
真正危险的不是“冷”，而是湿冷和停下后的失温风险。

### 风大
风不仅影响体感温度，也会影响暴露路段、山脊和营地安全。

### 雨天
雨并不一定等于不能走，但会显著提高湿滑、失温、视野下降和装备失效的概率。

## 实用判断原则

- 看天气不只看一个 App
- 看整段时间，而不是只看出发时刻
- 把最差情况也纳入准备
- 恶劣天气优先取消或降级计划

## 很值得养成的习惯

每次走完一条线后，回看当天天气与实际体感的差异。长期积累后，你会比单纯看预报更会判断自己适合什么天气窗口。
