# 户外的底层认知

## 为什么做户外

- 接触自然，恢复情绪与注意力
- 提升体能、耐力与长期行动能力
- 训练风险判断和自我照顾能力
- 建立对身体、天气、路线和节奏的真实感知

## 户外不只是“出去玩”

户外是一套系统化活动。体验质量往往不取决于冲动出发，而取决于准备质量：体能、天气、路线、装备、节奏和撤退判断都算在内。

## 三条核心原则

- 安全优先，先问能不能安全回来，再问能不能走得更远
- 量力而行，路线选择要匹配当前体能、经验和天气窗口
- 计划先于出发，把关键风险留在出发前解决

## 常见误区

- 只看风景，不看风险
- 只买装备，不练体能
- 只看攻略，不做路线判断
- 把一次成功，误以为长期稳定能力
