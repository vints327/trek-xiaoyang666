# 户外知识体系

这部分是从 Obsidian 中的“户外知识体系”整理出来的发布版。目标不是展示所有碎片笔记，而是提炼一套适合公开阅读、长期维护、逐步扩写的主线内容。

## 这套体系解决什么问题

很多人进入户外的方式，是先被风景吸引，再被装备种草，最后在路线、天气、节奏或安全判断上吃亏。真正重要的不是一次冲动出发，而是逐步建立稳定能力：会选路线、会看天气、会管理体能、会控制风险。

所以这套体系把内容拆成九块：底层认知、路线规划、装备补给、天气判断、导航信息、过夜系统、团队协作、安全风险，以及面向不同人群的实践建议。它既适合刚入门的人，也适合已经开始走线、想建立长期方法论的人。

## 推荐阅读路径

1. [底层认知](/outdoor/basics/)：先理解为什么做户外，以及什么叫“长期可持续的户外能力”
2. [路线与规划](/outdoor/planning/)：学会出发前该怎么判断路线和节奏
3. [装备与补给](/outdoor/gear/)：把装备放回场景里，而不是陷入参数焦虑
4. [天气判断](/outdoor/weather/) 与 [导航与信息判断](/outdoor/navigation/)：这是很多新手的薄弱环节
5. [过夜与露营](/outdoor/overnight/) 与 [团队协作](/outdoor/teamwork/)：适合进入更完整的户外场景前阅读
6. [安全与风险](/outdoor/safety/)：不是用来吓人，而是用来建立边界感
