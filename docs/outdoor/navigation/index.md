# 导航与信息判断

## 导航不是“会看轨迹”就够了

很多人把导航理解成打开轨迹跟着走，但真正的导航能力，至少包括地图理解、离线准备、路标识别、现场判断，以及当轨迹和现实不一致时如何处理。

## 基础能力

- 会用离线地图
- 会看海拔变化和大致地形
- 会识别路标和岔路
- 知道信号丢失时怎么处理
- 不过度依赖单一设备

## 信息判断为什么重要

公开攻略、短视频分享和路线记录经常会省略很多前提条件，比如天气、队伍配置、体能基础、季节差异。你看到的是别人走通的结果，不是你一定能复制的过程。

## 实用原则

- 一条路线至少交叉看多份信息
- 对“轻松”“适合新手”“风景绝美”这类描述保持警惕
- 判断路线时，把自己放进去，而不是把自己投射成作者

## 出现问题时先做什么

- 先停下来，不要边慌边乱走
- 回到最近确定的位置
- 检查离线地图、电量和队伍状态
- 如果判断条件不足，优先回撤而不是硬找捷径
