# 过夜与露营

## 过夜系统的核心不是“露营氛围感”

一旦从一日线进入过夜场景，准备逻辑就会明显变化。你不再只是考虑白天怎么走，还要考虑晚上怎么住、怎么吃、怎么保暖、怎么处理湿衣服和垃圾。

## 住的系统

- 帐篷、天幕、防潮垫、睡袋是核心
- 住得稳，第二天的恢复才有保障
- 营地选择要看地势、排水、风口和安全性

## 吃的系统

- 燃料够不够
- 热量够不够
- 炉具和炊具是否匹配
- 洗漱、清洁和垃圾处理是否有方案

## 什么时候适合开始过夜

- 你已经有稳定的一日徒步经验
- 你对天气、保暖和补给有基础理解
- 你能接受重量增加后的体能变化
- 你知道一旦状态不对，要比一日线更早做撤退判断

## 一条很实用的建议

第一次过夜不要追求画面感，优先选成熟营地、低技术难度和良好天气窗口，把“能稳稳完成”放在第一位。
