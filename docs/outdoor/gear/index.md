# 装备与补给

## 装备要回到场景里谈

“买什么”不应该是最先问的问题，先问“我要去什么场景”。城市近郊短徒步、全天暴晒山路、多日过夜线、雨季山线，对装备的需求完全不同。

## 装备原则

- 先够用，再升级
- 按场景选装备，而不是按冲动买装备
- 重量和安全永远要平衡
- 不要靠堆装备掩盖判断能力不足

## 穿着系统

### 内层
以速干为主，负责排汗与舒适，不建议在高强度场景穿纯棉。

### 中层
抓绒或轻量保暖层，负责保温，尤其是在停下来和早晚温差明显的场景里很关键。

### 外层
防风防雨层，决定你在风、雨、低温和湿滑环境中的下限。

## 核心基础装备

- 背包
- 水与补给
- 头灯和备用电源
- 防晒与保暖用品
- 简单应急药品
- 导航信息与离线地图

## 补给原则

- 水要少量多次补，不要等明显口渴
- 高温和长距离更要关注电解质
- 以易消化、稳定释放能量的食物为主
- 过夜线要把热量、燃料和补给点一起算
