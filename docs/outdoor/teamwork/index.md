# 团队协作

## 团队问题通常不是出发后才发生

大部分团队矛盾，其实在出发前就能看出来：体能差异、节奏预期、风险偏好、补给准备、拍照需求、撤退判断，如果不提前讲清楚，路上就会不断放大。

## 结伴的价值

- 提高安全性
- 分担信息、装备和应急能力
- 降低在陌生路线上的不确定性
- 在疲劳时提供更稳定的判断

## 团队里最容易出问题的点

- 体能差异过大
- 节奏和停留习惯不一致
- 装备准备不一致
- 关键节点没人负责决策
- 明明该回撤，却因为面子拖延

## 出发前要说清楚什么

- 路线与返程时间
- 谁带导航信息
- 谁带公共装备
- 出现不适时如何处理
- 哪些条件触发回撤

## 一个团队原则

团队速度取决于最慢的人，团队安全取决于最谨慎的人。
