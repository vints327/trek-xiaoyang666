# 安全与风险

## 安全不是恐惧，而是边界

真正成熟的户外能力，不是敢不敢上，而是知道边界在哪里。很多事故并不是因为一个巨大错误，而是多个小失误叠加：低估天气、轻视补水、忽略疲劳、错过回撤时机。

## 常见风险

- 迷路与轨迹偏离
- 脱水与能量崩盘
- 失温和中暑
- 扭伤、摔伤和滑坠
- 情绪上头后的错误决策

## 风险管理方法

- 提前报备路线
- 准备离线地图和应急方案
- 设定明确回撤条件
- 保持通讯与定位能力
- 在天气和状态变差时优先降级计划

## 应急认知边界

基础急救知识很重要，但更重要的是知道自己只能做基础处理。遇到持续恶化、高海拔明显异常、无法继续行走或意识状态变化时，判断和求助优先于硬撑。
