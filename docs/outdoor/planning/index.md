# 路线与规划

## 出发前要看什么

- 总距离与累计爬升
- 路况类型与技术难度
- 天气、风、降雨和体感温度
- 补给点、撤退点和通讯条件
- 同行人的体能与节奏是否匹配

## 行程设计原则

- 时间预估宁可保守，不要压满
- 给天气和意外留缓冲
- 设定明确的返程截止时间
- 不在疲劳时做极限决策

## 如何选适合自己的路线

- 新手优先从低海拔、短距离、低技术路线开始
- 先积累稳定的连续行走能力，再碰更长线路
- 海拔、负重、连续天数不要同时拉满

## 计划清单

- 确认路线 GPX、离线地图和备选方案
- 报备出发时间、路线和回程时间
- 检查天气窗口与降雨风险
- 估算补水与补给需求
