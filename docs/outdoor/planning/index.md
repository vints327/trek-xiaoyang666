# 路线与规划

## 规划比装备更先决定体验

很多路线的体验差异，并不是因为装备差距，而是因为规划质量不同。路线选错、节奏安排过满、回撤窗口没设好，就算装备不错，也很难走得舒服。

## 出发前至少要看五件事

### 总距离与累计爬升
不要只看公里数。很多山线公里数不长，但累计爬升大，强度会远高于平路徒步。

### 路况类型与技术难度
石阶路、碎石坡、泥地、雨后湿滑、暴晒无遮阴，这些因素对体能消耗和风险的影响都很大。

### 补给点与撤退点
成熟路线和野路最大的差别之一，就是补给和撤退的可行性。真正能减少风险的，往往不是更强，而是更早知道哪里能补、哪里能撤。

### 天气窗口
路线要和天气一起看。晴天可走，不等于雨天也能走；体感温度、风、湿度和能见度都要一起判断。

### 同行队伍匹配度
团队速度取决于最慢的人。体能、经验、对风险的容忍度，如果差异太大，出发后一定会产生摩擦。

## 行程设计四条原则

- 时间预估宁保守，不压满
- 高强度日不要连续堆叠
- 给突发情况留出缓冲
- 设好返程截止时间和撤退条件

## 适合新手的路线选择逻辑

- 从低海拔、短距离、低技术性开始
- 先建立连续行走能力，再增加负重和海拔
- 不同时把距离、爬升、天气难度全部拉高

## 一份简化的出发前检查

- 路线是否匹配当前体能
- 天气是否在可接受范围内
- 是否有离线地图和备用方案
- 是否告诉过别人你的路线和回程时间
- 是否知道在哪些节点必须果断下撤
